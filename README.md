# Home Tour Builder Agent

An LLM-powered home tour planning agent that turns NYC rental listings into a ranked shortlist, tour route, broker outreach messages, and viewing checklist.

## Current Constraints (v1)
- Max budget: $4,800
- Min: 2 bed / 1 bath
- Neighborhoods: West Village, Chelsea, UES, UWS, Williamsburg, Boerum Hill, Cobble Hill, Park Slope
- Must-have: dishwasher or space for dishwasher
- Dealbreakers:
  - Long walk to subway (>10 min)
  - Any room without windows
  - Walk-up above one flight
